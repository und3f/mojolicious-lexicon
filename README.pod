=pod

L<Mojolicious> lexicon generator for L<Mojolicious::Plugin::I18n>.
It search for I18N lexemes in L<Mojolicious> template and generates
lexicon file.

=head2 Usage example

./script/my_mojolicious_app generate lexicon
